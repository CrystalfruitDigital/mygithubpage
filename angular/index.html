    Section I: Set Up Basics

    Feel free to skip this section if you're already familiar with creating a basic project outline

    What is Angular?

    Angular is web app creation simplified. It takes object creation and manipulation to a whole new level of easy and helps you organize and manage data in your code.

    In this overview, I will be walking you through on how to start learning Angular and how it can be very useful.

    Requirements

    - Basic HTML/CSS/JS Knowledge
    - General unstanding of Object Oriented Programming

    Example One

    Angular Data Binding

    {{ngTest}}


    The input box is a "model" named "ngTest". The text paragraph that changes with the input is an "Angular binding" named "ngTest". Behind the scenes, AngularJS links the two objects together and dynamically updates the binding with the model's value.

    That was a lot of jargon but don't worry, I'll be breaking it down later.

    Setting Up

    Create a new folder and start a new project in your favorite text editor! For this walkthrough I'll be using Sublime Text. Feel free to use whatever you'd like!

    Step 1:

    Create a new group of index.html, styles.css and main.js files in your project folder.

    < index.html > 
    <html>

  <header>
    <title>My First Angular App</title>

    <!-- CSS files go here -->

  </header>

  <body>
    <h1>Hello World!</h1>

    <!-- Scripts go here -->

  </body>

</html>

    < styles.css > 
    @import url(http://fonts.googleapis.com/css?family=Roboto);

html, body{
  margin: 0;
  padding: 0;
  box-shadow: inset 0px 0px  40px -20px black;
  font-family: 'Roboto';
}

    < main.js > 
    // Angular code goes here...

    Import jQuery and Angular

    Place jQuery into your HTML document, then paste in the Angular CDN. Also, link your styles.css and main.js files while you're at it!

    <!-- Insert in header -->
<link rel="stylesheet" href="styles.css"/>

<!-- Insert at bottom of body -->
<script src="https://code.jquery.com/jquery-2.1.4.min.js"></script>
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.3.14/angular.js">
</script> 
<script src="main.js"></script>

    Importing these links into your HTML document will give you access to jQuery and AngularJS and link your index.html with your main.js and styles.css files (assuming you created them in the same folder as the HTML).



    Section II: Getting Started with Angular

    Feel free to skip this section if you know how to set up an Angular project already

    Getting Started

    Now that everything is in place, let's turn this project into an Angular app!

    <!-- Add "ng-app='yourAppName'" -->
<html ng-app="myApp">

<!-- Add "ng-controller='ctrlName'" -->
<body ng-controller="MainController">

    - Using the ng-app identifier, we can declare the Angular.js app that the .html file will be linked to.
    - Using the ng-controller identifier, we can declare the section of the .html file that will use Angular. 



    var app = angular.module("myApp", []);
app.controller('MainController', ['$scope', function($scope){
	// Insert code here!
}]);

    - Let's get into our Javascript file now - we'll be declaring a new variable (in this case, 'app') and we'll be using it to initialize a new angular module (in this case, 'myApp')

    - The square brackets that are included in the parameters function as space to insert Angular add-ons, such as Angular resource and such.

    - We call the "controller" function on our new 'app' variable, which links to the controller with the parameter name in the .html file (in this case, 'MainController')

    - '$scope' is what Angular uses to define objects within the controller's domain - basically anything within the domain (defined by tags that start with 'ng-' such as 'ng-model' and 'ng-binding') can be controlled using $scope.objectName
